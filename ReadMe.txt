把AI_Microwave_Oven解压到D盘根目录下
安装python 3.8.5
安装anaconda3 (https://www.anaconda.com/products/individual#Downloads)
安装pycharm
配置环境到pycharm
利用pip命令安装 cv2   (pip install opencv-python)
利用PyAudio   （pip install PyAudio）（确保有收音设备）
确保有摄像头
(连接温度传感器，若无连接，显示传感器未连接，需连接特定的传感器(use特定传感线，Dci红外温度传感器))
运行 app/main.py
